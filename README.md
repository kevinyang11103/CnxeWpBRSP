# 项目简介

## 核心功能
- 用户查询二手商品信息
- 用户查询求助信息
- 用户浏览校园资讯
- 用户兑换积分
- 用户查看我的订单
- 管理员管理所有用户信息

## 适用场景
- 校园内二手物品交易
- 校园内求助信息发布与查询
- 校园资讯获取
- 用户积分兑换服务

## 技术特点
- 采用Spring Boot框架，提供稳定的服务
- 使用Vue.js实现前端界面，提升用户体验
- 结合前后端分离架构，提高开发效率
- 提供便捷的用户管理和信息发布功能
